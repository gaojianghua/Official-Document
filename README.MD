<h3 align="center">开源可自定义导航的导航站平台</h3>

## 说明

Wolffy，是基于[Next.js](https://www.nextjs.cn)框架与[Ant-Design](https://ant-design.gitee.io)UI库制作的可自定义导航链接和印记卡片的导航站平台。在工作中可方便快捷打开您想访问的网站。
